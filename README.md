# vigIA — Documentação do PBL

Documentação técnica do projeto vigIA, desenvolvido na disciplina FGA0083 — Aprendizado de Máquina | UnB 2026-1 | Grupo 4.

## Acessar o github pages

Para uma melhor visualização da documentação, considere acessar: https://felipejrdev.github.io/vigIA-Goias/

## Rodar localmente

```bash
pip install -r requirements.txt
mkdocs serve
```

Acesse `http://127.0.0.1:8000`.

## Branch

Esta branch (`docs`) contém apenas a documentação. O código-fonte do projeto está na branch `main`.
